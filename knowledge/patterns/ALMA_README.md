# DOME Brain - Patterns

This directory contains design patterns and workflows extracted from experience.

## Structure
Each pattern is stored as a Markdown file with:
- **Definition**: What the pattern is
- **Details**: How to apply it, when to use it
- **Related Tags**: Keywords for cross-referencing

## Example
```markdown
# Pattern: Client Utility Coordination

## Definition
A workflow pattern for coordinating service upgrades between client and utility.

## Details
1. Identify utility based on location
2. Send client instructions
3. Wait for confirmation
4. Generate work packet

## Related Tags
`workflow`, `service-upgrade`, `coordination`
```
